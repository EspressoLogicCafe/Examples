Listener

Listeners use an existing connection to listen to a route. When a Message arrives along the route endpoint configured by a parameter,the listener code is triggered and the below parameters are injected into the JavaScript scope

  • payloadAsString
  • exchangeException
  • headers
  • outHeaders
  • properties
  • exchange
  • message
  • typeConverter
